# Database Schema

## Users
- `id` (serial)
- `email` (varchar)
- `password_hash` (varchar)
- `role` (varchar)
- `name` (varchar)
- `company_id` (int)
- `created_at` (timestamp)
- `updated_at` (timestamp)

## Companies
- `id` (serial)
- `name` (varchar)
- `industry` (varchar)
- `address` (text)
- `timezone` (varchar)
- `created_at` (timestamp)
- `updated_at` (timestamp)

## Employees
- `id` (serial)
- `employee_id` (varchar)
- `first_name` (varchar)
- `last_name` (varchar)
- `email` (varchar)
- `phone` (varchar)
- `department` (varchar)
- `position` (varchar)
- `manager_id` (int)
- `status` (varchar)
- `hire_date` (date)
- `salary` (numeric)
- `bank_account` (varchar)
- `tax_id` (varchar)
- `emergency_contact` (jsonb)
- `documents` (jsonb)
- `created_at` (timestamp)
- `updated_at` (timestamp)

## Attendance
- `id` (serial)
- `employee_id` (int)
- `date` (date)
- `clock_in` (timestamp)
- `clock_out` (timestamp)
- `break_minutes` (int)
- `status` (varchar)
- `notes` (text)
- `created_at` (timestamp)
- `updated_at` (timestamp)

## Leaves
- `id` (serial)
- `employee_id` (int)
- `leave_type` (varchar)
- `start_date` (date)
- `end_date` (date)
- `days_requested` (numeric)
- `status` (varchar)
- `approver_id` (int)
- `notes` (text)
- `created_at` (timestamp)
- `updated_at` (timestamp)

## Payroll Records
- `id` (serial)
- `employee_id` (int)
- `pay_period_start` (date)
- `pay_period_end` (date)
- `basic_salary` (numeric)
- `overtime_pay` (numeric)
- `bonuses` (numeric)
- `allowances` (numeric)
- `tax_amount` (numeric)
- `insurance_deductions` (numeric)
- `loan_deductions` (numeric)
- `net_pay` (numeric)
- `status` (varchar)
- `created_at` (timestamp)
- `updated_at` (timestamp)

## Audit Logs
- `id` (serial)
- `user_id` (int)
- `entity` (varchar)
- `action` (varchar)
- `details` (jsonb)
- `ip_address` (varchar)
- `created_at` (timestamp)
