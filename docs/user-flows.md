# User Flow Diagrams

## Super Administrator

1. Log in
2. Access system settings
3. Manage companies and users
4. Configure roles and permissions
5. Review audit logs

## HR Administrator

1. Log in
2. View employee directory
3. Add or update employee records
4. Approve leave requests
5. Manage performance reviews

## Payroll Administrator

1. Log in
2. Review payroll schedule
3. Run payroll calculations
4. Generate payslips
5. Review payroll history and reports

## Department Manager

1. Log in
2. Check team attendance
3. Approve or deny leave
4. Complete performance reviews
5. Download department reports

## Employee

1. Log in to employee portal
2. View personal profile
3. Submit leave request
4. Download payslips
5. Review attendance logs
